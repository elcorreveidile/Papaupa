# Prompts para generar fondos de página con ChatGPT (imágenes)

## Cómo usarlos
1. En ChatGPT, pega uno de los prompts y pide la imagen.
2. Pídela en **horizontal 16:9** (p. ej. 1920×1080) para escritorio. Si quieres,
   pide también una versión **vertical 9:16** para móvil.
3. Descárgala y guárdala en el repo como el archivo de esa página:
   - Inicio → `public/images/fondo-inicio.jpg`
   - Menú → `public/images/fondo-menu.jpg`
   - Juegos → `public/images/interior-juegos.jpg` (o dime y lo cambio)
   - (puerta de entrada → `public/images/fachada.jpg`)
4. Reemplaza el archivo (mismo nombre) y recarga. Si tu imagen sale muy clara o
   muy oscura, dime y ajusto el "velo" (transparencia) de cada página.

## Reglas que conviene incluir SIEMPRE
- "sin texto, sin letras, sin logotipos" (que no escriba palabras).
- "pensado como fondo de web, tonos apagados/desvaídos, con zonas tranquilas
  para poder leer texto encima".
- "estética retro años 70, cálida".
- Paleta: crema `#f5ecd7`, mostaza `#f2b705`, terracota `#c4502d`, granate
  `#8c2f1f`, oliva/marrón, verde azulado. Toques de Colombia (plátano, café).

---

## 1) Inicio (home)
```
Crea un fondo para una página web de un restaurante colombiano-mediterráneo
con estética retro de los años 70. Patrón de papel pintado vintage suave
(formas onduladas tipo "ogee", lozenges) en tonos crema, mostaza, terracota y
oliva apagado. Aspecto desvaído y elegante, como pared de un café antiguo.
Sin texto ni logotipos. Pensado como fondo de web: tonos suaves y zonas
tranquilas para leer texto encima. Horizontal 16:9, alta resolución.
```

## 2) Menú (la carta)
```
Fondo para la carta de un restaurante colombiano-mediterráneo, estética retro
años 70. Mantel de cuadros vichy en crema y mostaza muy suaves, o madera cálida
de mesa antigua con una insinuación de plátanos verdes, café y especias en las
esquinas, desenfocado. Aspecto apetecible pero apagado, para poner texto encima.
Sin texto ni letras. Horizontal 16:9, alta resolución, tonos cálidos desvaídos.
```

## 3) Juegos (el patio)
```
Fondo divertido y retro para una zona de juegos infantil de un restaurante
colombiano. Papel pintado años 70 con un patrón geométrico alegre (círculos
concéntricos naranjas, mostaza y marrón, estilo setentero), muy suave y
desvaído. Ambiente cálido y familiar. Sin texto ni personajes. Pensado como
fondo de web con zonas tranquilas. Horizontal 16:9, alta resolución.
```

## 4) Puerta de entrada (alternativa a la fachada)
```
Fondo a pantalla completa, atmosférico, de la entrada de un pequeño restaurante
retro en un callejón del barrio del Realejo (Granada): piedra cálida, puerta de
madera con cortina verde, luz interior acogedora al atardecer. Estética
cinematográfica y nostálgica, tonos cálidos. Sin texto ni logotipos. Vertical
9:16 para móvil y horizontal 16:9 para escritorio. Alta resolución.
```

## 5) Genérico (cualquier página)
```
Textura de fondo sin costuras (seamless) de papel pintado retro de los años 70
para una web de restaurante. Patrón sutil y repetitivo en crema, mostaza y
marrón cálido, muy desvaído, casi monocromo. Sin texto. Pensado para poner
contenido encima sin perder legibilidad. Cuadrado, alta resolución.
```

---

**Consejo:** si quieres que un fondo se vea MUY protagonista (foto a pantalla
completa), genérala con una "zona más clara" donde irá el texto, o dime y pongo
el contenido sobre tarjetas semitransparentes para que la imagen se vea entera
y el texto siga legible.
